# FoodAnalysis_limpopo_study
#  Food Safety Data Analysis (South Africa)

## Project Overview

This project investigates microbial contamination in ready-to-eat foods sold in Thohoyandou, Limpopo. By combining laboratory knowledge with data analytics, the goal is to uncover hidden risks in everyday food consumption.

---

##  Problem

Foodborne illnesses remain a major public health concern. While food may appear safe, microbial contamination can exist due to poor hygiene and handling practices.

---

## Data Source

This project is based on an open-access South African study:

**Microbial Safety of Ready-to-Eat Food Sold by Retailers in Thohoyandou, Limpopo Province**
Source: ResearchGate
https://www.researchgate.net/publication/369549107

---

## Tools Used

* R (RStudio) → statistical analysis
* Power BI → dashboard visualization
* Google Sheets 

---

## Key Insights

* Sandwiches showed the highest contamination levels
* Retailer 4 had consistently higher bacterial counts
* E. coli presence suggests potential hygiene failures
* Acceptable total counts may still hide dangerous pathogens

---

## Recommendations

* Improve hygiene training for food handlers
* Strengthen inspection of high-risk food types
* Use data monitoring for early detection
